About Project

This project, officially titled “Solar Assisted Electric Multipurpose Agricultural Vehicle”, is developed as a final year diploma project by Mechanical Engineering students of Dibrugarh Polytechnic, Assam. We have named our system “KRIXHI MATE” as a simple and recognizable identity for the project. It represents a practical, eco-friendly solution designed to support modern agricultural activities using solar-assisted electric power.

Our Main Features

📡 Wireless Control – Control via mobile app
🌱 Seed Sowing – Automatic mechanism
💧 Spraying – Uniform pesticide spraying
